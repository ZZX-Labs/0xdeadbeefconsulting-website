Canonical Project Data

One Manifest System Across Every Website

Project information is loaded from the canonical ZZX-Labs project index and categorized project manifests. Updates made to those manifests can automatically propagate to the consulting software, project, and portfolio catalogs.

Canonical Root Manifest

The global project index is loaded from:

https://zzx-labs.io/projects/manifest.json

Category Manifests

Category data is loaded from paths such as:

https://zzx-labs.io/projects/software/manifest.json

Project Manifests

Individual project metadata may be loaded from:

https://zzx-labs.io/projects/software/example/manifest.json
Cross-origin requirement: zzx-labs.io must permit browser access from https://0xdeadbeefconsulting.io through an appropriate Access-Control-Allow-Origin response header.
