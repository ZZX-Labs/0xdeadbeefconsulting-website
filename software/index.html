Our Open Source Software

Modular. Auditable. Battle-tested. Free to use, fork, and build upon.

RedPanda

A source-code obfuscator and deterrence system targeting hostile actors, leveraging language-based deterrence, cryptographic verification, and hash-based validation layers.

View on GitHub


Bit-Tick

A high-frequency Bitcoin price ticker using real-time APIs from top global exchanges, excluding aggregator distortion. Dark mode, layered unit support, and self-hostable.

Live Demo / Source


Cryptainer

A multi-layered encryption GUI powered by GPG, AES, RSA, and a custom virtual cryptographic pet (GKS) guarding Bitcoin wallets in secure containers.

Explore Cryptainer


AudioSieve

A Python-based music library manager using VLC, Mutagen, FFmpeg, and PyQt5. Tag, play, edit, and organize audio libraries with CLI/GUI hybrid support.

Browse Code


YT/RP + Soundclouddler

Command-line audio/video downloaders using yt-dlp with custom cookie, VPN, and proxychain support. CLI-first. Built for speed, reliability, and censorship evasion.

YT/RP Suite


GGAI / CGAI / GP

Custom GPT-powered local assistants for specialized tasks: generalist reasoning (GGAI), creative generation (CGAI), and personal device automation (GP).

Launch GGAI


MetaTagDB

Full-stack metadata extraction, tagging, and library management for video files, using hashes, screenshots, and AI for deduplication and indexing.

View MetaTagDB
